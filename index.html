Getting Started Documentation

Blocks

Similar to the official FIRST apps, VRS Simulator offers support for a scratch-like visual programming language. FTC Programmers who use Blocky will feel right at home!

Code

VRS integrates a similar solution to FIRST’s official OnBot Java solution. FTC developers familiar with FTC Java will feel comfortable!

Play

Plug in your Dualshock or f310 controllers and practice driving your robots in real-time with real controls!

Open Source

VRS is developed by FIRST students and alum, helping ensure the continuity of the project
